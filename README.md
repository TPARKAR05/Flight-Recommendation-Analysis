# Flight Recommendation Analysis

## Project Overview
This project focuses on developing a flight recommendation system utilizing various data analysis techniques and machine learning algorithms. The goal is to provide users with optimal flight options based on their preferences and historical flight data.

## Technologies Used
- **HTML**: 57.9% - Used for the front-end interface to display flights and recommendations.
- **Python**: 42.1% - Utilized for data processing, analysis, and machine learning model development.

## Features
- User-friendly interface for inputting flight preferences.
- Real-time analysis of available flights.
- Machine learning algorithms to predict optimal flight recommendations based on user data.

## Installation
To set up this project locally, follow these steps:
1. Clone the repository:  
   ```bash
   git clone https://github.com/TPARKAR05/Flight-Recommendation-Analysis.git
   ```
2. Navigate to the project directory:  
   ```bash
   cd Flight-Recommendation-Analysis
   ```
3. Install required packages:  
   ```bash
   pip install -r requirements.txt
   ```

## Usage
To run the application:
```bash
python main.py
```
